# images フォルダ

ここに下記のファイル名（すべて **WEBP形式**）で画像を保存すると、LPの該当箇所に自動で表示されます。
ファイル名はLP側と一致させてください（大文字・小文字も同じに）。画像が未設置のスロットは、
ファイル名とサイズが書かれたプレースホルダが表示されます。

※ファーストビューは画像ではなく「動くアプリデモ」に変更したため、FV用の画像は不要になりました。

| ファイル名 | サイズ(px) | 内容 |
|---|---|---|
| `problem-flood.webp` | 900×675 | 情報の洪水に困っている人のイラスト |
| `app-ui-next.webp` | 1280×800 | アプリ画面①：NEXT BTCボタンの画面 |
| `app-ui-list.webp` | 1280×800 | アプリ画面②：類似率順に30銘柄が並ぶ画面 |
| `app-ui-warning.webp` | 1280×800 | アプリ画面③：警告マークと理由の解説画面 |
| `demo-poster.webp` | 1600×900 | デモ動画のサムネイル |
| `step-install.webp` | 720×540 | STEP1：インストールのイラスト |
| `step-view.webp` | 720×540 | STEP2：画面を見るイラスト |
| `step-decide.webp` | 720×540 | STEP3：自分で判断するイラスト |
| `devices-lineup.webp` | 1400×700 | Windows・iPhone・Androidが並んだ画像 |
| `stance-team.webp` | 1200×800 | データと向き合う運営チームのイメージ |
| `cta-app.webp` | 1200×900（透過推奨） | 最終CTA横のアプリビジュアル |

## 作成のコツ

- ページの配色に合わせ、**白背景＋ブルー(#1B6FD0)＋ゴールド(#E5B94E)** のトーンで揃えると統一感が出ます。
- `app-ui-*` の3枚は、実物のアプリのスクリーンショットがあればそちらの方が信頼感が出ます。
